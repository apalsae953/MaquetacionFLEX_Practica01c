# Maquetación Flex - Antonio Palacios Sáez

## Descripción del proyecto
Este proyecto es una práctica de maquetación web con Flex.
El objetivo principal es demostrar el uso de Flex para organizar elementos en una página web, creando una estructura limpia y ordenada que incluye una barra de navegación y una galería de imágenes con pies de foto.

## Contenido de la publicación
El repositorio contiene la estructura completa del proyecto web:
*   index.html: La estructura semántica de la página.
*   style.css: Hoja de estilos donde se definen las reglas de Flexbox y el diseño visual.
*   imagenes/: Carpeta que contiene los recursos gráficos utilizados en la galería.

## Desarrollo del proyecto
El desarrollo se ha centrado en la creación de un layout flexible:
*   Se ha utilizado display: flex para gestionar la distribución de los elementos contenedores
*   La página se divide en un encabezado, una sección principal con dos filas de contenido y un pie de página.
*   Se han aplicado estilos para asegurar que las tarjetas de imágenes y el menú de navegación se alineen correctamente.
