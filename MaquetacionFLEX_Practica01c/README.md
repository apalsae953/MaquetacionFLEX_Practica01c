# Maquetación Flex - Antonio Palacios

## Descripción del proyecto
Este proyecto es una práctica de maquetación web desarrollada para la asignatura de **Diseño de Interfaces Web (DIW)** del curso de 2º de Desarrollo de Aplicaciones Web (DAW).

El objetivo principal es demostrar el uso de **CSS Flexbox** para organizar elementos en una página web, creando una estructura limpia y ordenada que incluye una barra de navegación y una galería de imágenes con pies de foto.

## Contenido de la publicación
El repositorio contiene la estructura completa del proyecto web:
*   `index.html`: La estructura semántica de la página.
*   `css/style.css`: Hoja de estilos donde se definen las reglas de Flexbox y el diseño visual.
*   `imagenes/`: Carpeta que contiene los recursos gráficos utilizados en la galería.

## Desarrollo del proyecto
El desarrollo se ha centrado en la creación de un layout flexible:
*   Se ha utilizado `display: flex` para gestionar la distribución de los elementos contenedores (`main`, `section.fila`, `nav`, `footer`).
*   La página se divide en un encabezado, una sección principal con dos filas de contenido (menú + imágenes) y un pie de página.
*   Se han aplicado estilos para asegurar que las tarjetas de imágenes y el menú de navegación se alineen correctamente.
